## Quick Start

For development, read docs in this order:
1. `01_PRD.md` — understand what you're building
2. `03_Architecture.md` — understand the system
3. `06_Database.md` — set up the database first
4. `11_Dev_Plan.md` — follow the hour-by-hour plan
5. `04_AI_Agents.md` + `10_Prompts.md` — implement the agents
6. `07_API.md` — build and test the API endpoints
