## Slide 1: Hook

> **"You don't need more courses. You need a plan."**

Most learners fail not because they're not motivated — but because they're overwhelmed by choice, scattered resources, and no clear path forward.

---

## Slide 2: Problem

Every self-learner has been here:

- 🔍 Googling "how to learn X" and getting 50 conflicting roadmaps
- 📚 Signing up for 3 courses and finishing none of them
- 💸 Paying for Udemy, Coursera, and YouTube Premium — still lost
- 📅 No structured daily plan — just a vague "I'll study sometime today"
- ❓ No way to test if you actually learned anything

**The result?** Skill gaps. Career stagnation. Wasted money. Frustrated learners.

---

## Slide 3: Solution

**SkillVerse** is an AI-powered learning platform that works like a personal AI tutor team.

You say: *"I want to become a backend developer in 3 months"*

SkillVerse responds with:
- ✅ A structured 12-week milestone roadmap
- ✅ Personalized daily tasks that fit your schedule
- ✅ Curated best-in-class resources per topic
- ✅ Hands-on capstone projects for each milestone
- ✅ Quizzes to reinforce retention
- ✅ A live progress dashboard

**One platform. One goal. Complete journey.**

---

## Slide 4: How It Works — The AI Agent Team

SkillVerse doesn't use a single AI — it uses a **coordinated team of 7 specialized agents**:

```
Your Goal 
  → 🎯 Goal Analyzer: "Backend dev in 12 weeks. Got it."
  → 📚 Curriculum Designer: "6 milestones. Let's build."
  → 📅 Daily Planner: "1 hour/day. Here's your schedule."
  → 🔍 Resource Finder: "Best free resources per topic."
  → 🔨 Project Mentor: "Build real projects."
  → ❓ Quiz Master: "Test your knowledge."
  → 📊 Progress Tracker: "You're 27% done. 🔥 5-day streak."
```

Each agent is a specialist. Together, they're your personal AI learning team.

## Slide 6: Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 14 (App Router) |
| Backend | Node.js + Express.js |
| Database | Supabase (PostgreSQL + Auth + RLS) |
| AI | Google Gemini 1.5 Pro |
| Deploy | Vercel (FE) + Railway (BE) |

**Key technical differentiator**: Multi-agent pipeline with structured JSON output. Each agent inherits the previous agent's output, creating a coherent, personalized learning system — not a generic prompt.
